# Bootstrap 页面标题（Page Header）

> 页面标题，会在网页标题四周添加适当地间距

> 使用页面标题（Page Header），需把标题放置在`.page-header`的`<div>`中

##### 示例
```html
<div class="page-header">
    <h1>
        周星驰的电影
        <small>大喜中透着大悲</small>
    </h1>
</div>
<p>大话西游&nbsp;&nbsp;&nbsp;--- &nbsp;&nbsp;&nbsp;看着看着就哭了</p>
```
