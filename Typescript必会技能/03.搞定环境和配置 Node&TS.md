# 搞定环境和配置 Node/TS

## ts-node

> `Node` 环境的 `typescript` 解释执行器

```
npm i -g ts-node
# yarn global add ts-node
```

### 配置文件： `tsconfig.json`

```
{
	"compilerOptions": {
		
	}
}
```

### `tsc (typescript compiler)`

> `ts` 的编译器

```
npm i -g tsc
# yarn global add tsc
```

### 和 `webpack` 一起使用

```
yarn add webpack ts-load typescript webpack-cli --dev
```
